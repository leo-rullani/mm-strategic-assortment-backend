# MM Flow redesign notes

Version: 1.0 visual rebrand

## Changed

- Added a new MM Flow logo family in light, dark and compact variants.
- Added a standalone visual override layer in `shared/css/mm_theme.css`.
- Rebranded titles, favicons and visible application headers.
- Redesigned login and registration as light form sheets on a graphite background.
- Redesigned buttons, inputs, dropdowns, checkboxes, dialogs and validation states.
- Redesigned dashboard cards, tables and chart colours.
- Redesigned board overview rows and Kanban cards.
- Added independent status colours for To-do, In progress, Review and Done.
- Updated PDF export branding without changing export behaviour.
- Replaced the old event marquee with strategic assortment modules.

## Intentionally unchanged

- JavaScript event handlers and element IDs
- Authentication storage keys
- API endpoint configuration
- CRUD behaviour
- Permission and member logic
- Task status values
- Comment handling
- Backend and database schema

## Content still requiring owner approval

The Privacy Policy and Imprint retain their existing legal content. Their header presentation is rebranded, but legal entity and contact information should only be changed after the final hosting and ownership setup is confirmed.
