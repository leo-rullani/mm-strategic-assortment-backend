board/board_templates.js

Provides small, framework-agnostic string template builders for: - Member profile circles (limited to 4 with "+X" overflow indicator) - Task detail person blocks (assignee/reviewer) - Single comment entries with optional delete button - Kanban task cards, including optional SFL Debriefing PDF export button - Move button group for changing task status Includes a few helpers for SFL board detection, safe HTML preview extraction, and PDF export.

board/pdf_export.js

Provides functions to open a print-ready popup with task HTML (user chooses "Save as PDF"), plus an optional board‑wide export via html2pdf. Includes helpers to read CSS variables from the host app and generate a self-contained CSS block for the PDF document. Exports: - exportDebriefingTaskPdf: open a new tab with app-like styling and trigger print - downloadDebriefingPdf: download an aggregated PDF for the whole board (html2pdf)

requests/requests.js

Provides a small in-memory/LocalStorage–backed request list to demonstrate rendering, filtering and inline status updates for assignments. Defaults: - Start: 15:00 - End: 20:00 - Location: "ROC Sissach" Status is optionally persisted under STORE_KEY in LocalStorage.
